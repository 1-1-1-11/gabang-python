# Claude Code 审查提示词

请只审查本任务 diff，不直接修改代码。

## 审查范围

- 只关注当前任务分支相对 `main` 的变更。
- 不评价未触及的历史代码，除非它直接影响本任务正确性。
- 如果发现阻塞问题，请给出最小可执行修复建议。

## 输出格式

请按以下分类输出：

```markdown
# Review: <任务编号>-<任务名>

## Blocker

- 无，或列出必须修复后才能合并的问题。

## Major

- 无，或列出高风险缺陷、接口不一致、测试缺口。

## Minor

- 无，或列出可后续处理的小问题。

## Question

- 无，或列出需要实现方确认的问题。
```

## 合并门禁

- 存在 Blocker 或 Major 时，不允许合并。
- Minor 和 Question 可以记录后合并，但需要在任务台账中说明处理决定。
